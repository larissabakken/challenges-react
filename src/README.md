 ### Challenges to learn React

Install the below tools/packages to run the project

#### Tools/packages
- [x] [NodeJS](https://nodejs.org/en/) >= 16.17.0
- [x] [NPM](https://www.npmjs.com/) >= 9.0.0

#### How to run the project
```bash
# Clone this repository
$ git clone

# Run the command
$ npm install

# Run the command
$ npm dev
```
